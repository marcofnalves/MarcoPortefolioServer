# MarcoPortefolioServer
This is the server component of the Marco Portfolio application, which serves portfolio data to clients.
## Features
- RESTful API to fetch portfolio data
- Built with Node.js and Express
- CORS enabled for cross-origin requests
- Modular route handling
- Error handling middleware

Environment variable configuration using dotenv
## Prerequisites
- Node.js (v14 or higher)
- npm (Node Package Manager)
- MongoDB (if using a database for portfolio data)
- Git (for version control)
- Postman or any API testing tool (optional, for testing the API)
- A code editor like Visual Studio 2022
